# Projet S6 Instrumentation

Ce dépôt contient le projet S6 d'instrumentation

Le projet est organisé en deux parties techniques basé sur le matériel hardware, chacune traitée
indépendamment pour garantir la clarté.

## Architecture du projet
- ESP32 : acquisition de données capteurs
- Raspberry Pi : traitement, serveur et supervision, SQL (stockage et gestion des données, Node-RED (automatisation et orchestration des flux)

Chaque partie est développée de manière indépendante.
